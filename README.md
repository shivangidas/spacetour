# SpaceSearch

### A NASA image search app

To run:

- npm install
- npm start
- http://0.0.0.0:4040

Features:

- User management
- Search images using keywords
- Open details in a modal
- Save/ Delete images
- Display saved images
- External API access (https://images.nasa.gov/docs/images.nasa.gov_api_docs.pdf)
- APIs for CRUD operations
- 404 handling
